# Product Idea

**Content Flow** is an AI-powered workspace that helps creators discover relevant content ideas, transform them into platform-specific content, and organize their weekly content workflow.

The product is designed for creators who want to publish consistently across multiple platforms but struggle with three recurring problems:

Finding relevant and timely topics
Adapting one idea to different social media platforms
Keeping track of drafts, scripts, and publishing tasks

Instead of using separate tools for research, writing, and planning, Content Flow combines these steps into one simple workflow:

Discover → Select → Generate → Refine → Organize
